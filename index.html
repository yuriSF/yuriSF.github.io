Software engineering portfolio

I am a professional linguist with experience in software engineering and natural language processing (NLP). Formerly a university professor, I developed a passion for technology when I was involved in a number of technical projects in academia: I mined dialect data from online mass media, developed an online dictionary for a project funded the Kansas Humanities Council, and built a digital infrastructure for the Writing Center at Fort Hays State University. Working on those projects, I learned all things Python from NLP to statistics to database management to web apps. To deepen my understanding of software engineering, I completed a full stack curriculum at Thinkful, with a focus on Javascript, jQuery, AJAX, RESTful APIs, Node.js, Flask, MongoDB, and PostgreSQL. Most recently, I have worked for Facebook as a computational linguist. I am now open for new opportunities in software engineering and/or natural language processing.

My Web Applications

Here you can find my linguistic web apps. Volga German Digitization is a site I built for the Kansas Humanities Council. It features a Python-powered cgi backend for a Volga German dictionary. I built the other apps as part of Thinkful's full stack curriculum. Gloss Maker is a RESTful API for the Yandex translation service. The other two apps have their own backends. Language Analyzer is powered by Node.JS and MongoDB, while Spanish Verb Conjugator by Python's Flask and PostgreSQL. Language Analyzer scrapes the content of a wesbite and returns a summary of basic NLP statistics for it. Spanish Verb Conjugator helps you learn Spanish verbs and their conjugations.
